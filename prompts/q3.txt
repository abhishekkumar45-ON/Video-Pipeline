## OUTPUT CONTRACT (strict)

- Every spoken line goes through `with self.voiceover(text="...") as t:` and animation
  run_times are driven by `t.duration` (audio-first timing). Never hardcode narration length.

## BRAND TOKENS (Kinematics / Orange Nelumbo)
- Background `#0E0D10`.
- IGNITION orange `#FF5A1F` (also `#FF7A2E`) = **the concept in focus**.
- VELOCITY CYAN `#3DE0D0` = **motion, vectors, measured data**.
- Carbon `#1E1B20` (panels), muted grey `#8A8A93`, error red `#E0483C`.
- HARD RULE: orange = concept-in-focus, cyan = motion/data. Never swap these roles.
- Fonts: use `JetBrains Mono` for equations/values/units and mono tags; it degrades
  gracefully if unavailable. Headlines can be bold Text.

## PERSISTENT CHROME (top of every scene)
- Top-left: `// <CHAPTER>` (small, muted-orange) + `ORANGE NELUMBO` sub-tag beneath it.
- Top-right: progress counter `NN / <total>` updating per beat (e.g. `03 / 06`).
- 1920x1080 frame; keep everything inside ~90% title-safe zone.

## NARRATION = TEACHING STYLE (critical — the voice is flat, the writing carries emotion)
- Write like a teacher talking, not a textbook. Short sentences.
- Use rhetorical cues: "Now, here's the clever move...", "Watch what happens.", "Look closely.".
- Create emphasis with pauses: use "..." and " — " so the voice breathes and lands key words.
  e.g. "The discriminant... is negative. And that — changes everything. No real solutions. None."
- Open with a hook, end by stating the final answer plainly.

## MATHTEX SAFETY (basictex only — avoid exotic LaTeX)
- Allowed: \frac, \tfrac, \mathrm, \sqrt, ^, _, greek, \cdot, \times, \le, \ge.
- FORBIDDEN inside MathTex: \boxed, \text, \begin{cases}, mhchem, exotic packages.
  For a boxed answer use a `SurroundingRectangle`, not \boxed. For units use \mathrm.
- Wrap any risky MathTex in try/except that falls back to plain `Text` if LaTeX fails.

## STRUCTURE (match q1.py)
- 5-7 beats: HOOK -> SETUP -> the physics/steps -> the key insight -> GRAPH/visual payoff
  -> boxed final ANSWER. Fade cleanly between beats. Update the progress counter each beat.
- Anchor the explanation to the KNOWN correct answer given in the task — never contradict it.
- Keep objects on-screen and non-overlapping (this is checked; off-screen/overlap = a bug).

## RENDERING FACTS (so your code is compatible)
- Rendered headless via: manim -ql --media_dir build/media scenes/<id>.py <ClassName>
- `mac_say_service.py` and `kokoro_service.py` are importable (project root is on PYTHONPATH).
- Do not read/write files, hit the network, or need any API key. Pure Manim + voiceover.

## GOLD EXAMPLE — study this, then write at this level:

"""
Orange Nelumbo · Kinematics · Q1 — "The Chase"
Two particles from A; does the second ever catch the first?
Answer: (D) never — discriminant < 0.

Brand: BG #0E0D10 · IGNITION orange = concept in focus · VELOCITY CYAN = motion/data.
Narrated via offline macOS `say` (mac_say_service.py must sit next to this file / on PYTHONPATH).

Render:
  source .venv/bin/activate
  eval "$(/usr/libexec/path_helper)"
  python batch_render.py --quality=-ql --drive=gdrive:JEE-Videos/Kinematics
"""

from manim import *
from manim_voiceover import VoiceoverScene
from kokoro_service import KokoroService

# ---------- brand tokens ----------
BG       = "#0E0D10"
IGNITION = "#FF5A1F"   # orange  — the concept in focus
EMBER    = "#FF7A2E"
CYAN     = "#3DE0D0"   # velocity cyan — motion, vectors & data
CARBON   = "#1E1B20"
TXT      = "#E8E6EC"
MUTED    = "#8A8A93"
REDX     = "#E0483C"

MONO = "JetBrains Mono"   # falls back gracefully if not installed


class Q1Kinematics(VoiceoverScene):
    def construct(self):
        self.camera.background_color = BG
        self.set_speech_service(KokoroService())   # af_heart — natural neural TTS (US)

        # ---------------- persistent chrome ----------------
        tag  = Text("// KINEMATICS", font=MONO, color=IGNITION).scale(0.32)
        mark = Text("ORANGE NELUMBO", font=MONO, color=MUTED).scale(0.26)
        tag.to_corner(UL, buff=0.4)
        mark.next_to(tag, DOWN, aligned_edge=LEFT, buff=0.12)
        prog = Text("01 / 06", font=MONO, color=MUTED).scale(0.26).to_corner(UR, buff=0.4)
        self.add(tag, mark, prog)

        def set_prog(n):
            new = Text(f"0{n} / 06", font=MONO, color=MUTED).scale(0.26).move_to(prog)
            self.play(Transform(prog, new), run_time=0.4)

        # ================= 01 — HOOK =================
        title = Text("THE CHASE", font=MONO, color=TXT, weight=BOLD).scale(1.1)
        sub   = Text("two particles, one straight line",
                     font=MONO, color=MUTED).scale(0.4).next_to(title, DOWN, buff=0.3)
        with self.voiceover(text="Two particles take off from the very same point — point A — "
                                 "along one straight line. So, here's our question... and it sounds simple. "
                                 "Does the second particle ever, actually, catch the first?") as t:
            self.play(Write(title), run_time=min(2.0, t.duration))
            self.play(FadeIn(sub, shift=UP*0.2), run_time=0.8)
            self.wait(max(0.1, t.duration - 2.8))
        self.play(FadeOut(title), FadeOut(sub))

        # ================= 02 — SETUP =================
        set_prog(2)
        line = Line(LEFT*5.5, RIGHT*5.5, color=MUTED).shift(DOWN*0.3)
        A = Dot(line.get_start(), color=TXT)
        Alab = Text("A", font=MONO, color=TXT).scale(0.5).next_to(A, DOWN, buff=0.2)
        p1 = Dot(color=CYAN).scale(1.3).move_to(line.get_start())
        p2 = Dot(color=EMBER).scale(1.3).move_to(line.get_start())

        c1 = MathTex(r"u_1=10\,\mathrm{m/s}", r"\quad a_1=2\,\mathrm{m/s^2}",
                     color=CYAN).scale(0.75)
        c2 = MathTex(r"u_2=20\,\mathrm{m/s}", r"\quad a_2=1\,\mathrm{m/s^2}",
                     color=EMBER).scale(0.75)
        c2b = Text("starts 5 s later", font=MONO, color=MUTED).scale(0.4)
        card1 = VGroup(Text("PARTICLE 1", font=MONO, color=CYAN).scale(0.4), c1).arrange(DOWN, buff=0.2)
        card2 = VGroup(Text("PARTICLE 2", font=MONO, color=EMBER).scale(0.4), c2, c2b).arrange(DOWN, buff=0.2)
        cards = VGroup(card1, card2).arrange(RIGHT, buff=1.4).to_edge(UP, buff=1.3)

        with self.voiceover(text="Particle one leaves first. Ten metres per second to start... "
                                 "and it keeps speeding up — accelerating at two metres per second squared.") as t:
            self.play(Create(line), FadeIn(A), Write(Alab), run_time=1.2)
            self.play(FadeIn(card1, shift=UP*0.2), FadeIn(p1), run_time=1.0)
            self.wait(max(0.1, t.duration - 2.2))

        with self.voiceover(text="Now — five whole seconds later — particle two sets off from that same point. "
                                 "It's faster off the mark, at twenty metres per second. But notice... "
                                 "a gentler acceleration. Just one.") as t:
            self.play(FadeIn(card2, shift=UP*0.2), FadeIn(p2), run_time=1.2)
            self.wait(max(0.1, t.duration - 1.2))

        # quick motion hint: slide the two dots a little
        with self.voiceover(text="Same direction, both of them. And your gut probably says — "
                                 "the faster one reels the first one in, right? Well... let's not guess. "
                                 "Let's turn this motion into equations.") as t:
            self.play(p1.animate.shift(RIGHT*2.2), run_time=1.4, rate_func=rate_functions.ease_in_out_sine)
            self.play(p2.animate.shift(RIGHT*1.6), run_time=1.0, rate_func=rate_functions.ease_in_out_sine)
            self.wait(max(0.1, t.duration - 2.4))
        self.play(FadeOut(line), FadeOut(A), FadeOut(Alab), FadeOut(p1), FadeOut(p2),
                  cards.animate.scale(0.85).to_edge(UP, buff=0.7))

        # ================= 03 — EQUATIONS =================
        set_prog(3)
        gen = MathTex(r"x = ut + \tfrac{1}{2}\,a t^2", color=MUTED).scale(0.85).shift(UP*0.6)
        e1  = MathTex(r"x_1 = 10t + t^2", color=CYAN).scale(1.0)
        e2  = MathTex(r"x_2 = 20(t-5) + \tfrac{1}{2}(t-5)^2",
                      color=EMBER).scale(1.0)
        note = Text("valid for  t ≥ 5", font=MONO, color=MUTED).scale(0.4)
        VGroup(e1, e2, note).arrange(DOWN, buff=0.5).shift(DOWN*0.4)

        with self.voiceover(text="Displacement, under constant acceleration, is u t plus one half a t squared. "
                                 "Hold on to that one.") as t:
            self.play(Write(gen), run_time=min(1.8, t.duration))
            self.wait(max(0.1, t.duration - 1.8))
        with self.voiceover(text="For particle one, that gives us — nice and clean — "
                                 "x one equals ten t, plus t squared.") as t:
            self.play(TransformFromCopy(gen, e1), run_time=min(1.6, t.duration))
            self.wait(max(0.1, t.duration - 1.6))
        with self.voiceover(text="But particle two — remember — has only been moving for t minus five seconds. "
                                 "So its position is twenty times t minus five... "
                                 "plus a half of t minus five, squared.") as t:
            self.play(Write(e2), run_time=min(2.2, t.duration))
            self.play(FadeIn(note), run_time=0.5)
            self.wait(max(0.1, t.duration - 2.7))
        self.play(FadeOut(gen), FadeOut(note), FadeOut(cards),
                  VGroup(e1, e2).animate.arrange(DOWN, buff=0.4).to_edge(UP, buff=1.1))

        # ================= 04 — CATCH-UP CONDITION =================
        set_prog(4)
        cond_lbl = Text("CATCH-UP CONDITION", font=MONO, color=IGNITION).scale(0.42)
        cond = MathTex(r"x_1 = x_2", color=IGNITION).scale(1.1)
        VGroup(cond_lbl, cond).arrange(DOWN, buff=0.25).next_to(VGroup(e1, e2), DOWN, buff=0.6)

        with self.voiceover(text="Now — when do they actually meet? Only when their positions are equal. "
                                 "So we set x one, equal to x two.") as t:
            self.play(FadeIn(cond_lbl), Write(cond), run_time=min(1.8, t.duration))
            self.play(Indicate(e1, color=IGNITION, scale_factor=1.08),
                      Indicate(e2, color=IGNITION, scale_factor=1.08), run_time=1.0)
            self.wait(max(0.1, t.duration - 2.8))
        self.play(FadeOut(e1), FadeOut(e2), FadeOut(cond_lbl),
                  cond.animate.to_edge(UP, buff=1.0))

        quad = MathTex(r"t^2 - 10t + 175 = 0", color=TXT).scale(1.1).next_to(cond, DOWN, buff=0.7)
        steps = MathTex(r"10t + t^2 = 20(t-5) + \tfrac{1}{2}(t-5)^2",
                        color=MUTED).scale(0.7).next_to(cond, DOWN, buff=0.55)
        with self.voiceover(text="Expand both sides, gather every term onto one line, multiply through by two... "
                                 "and watch what happens. The whole mess collapses — "
                                 "into one clean quadratic. t squared, minus ten t, "
                                 "plus one hundred seventy-five, equals zero.") as t:
            self.play(Write(steps), run_time=1.6)
            self.play(steps.animate.set_opacity(0.35).scale(0.8).next_to(quad, UP, buff=0.35),
                      run_time=0.8)
            self.play(Write(quad), run_time=min(1.8, max(0.6, t.duration - 3.0)))
            self.wait(0.2)
        self.play(FadeOut(steps), FadeOut(cond), quad.animate.to_edge(UP, buff=1.1))

        # ================= 05 — DISCRIMINANT =================
        set_prog(5)
        disc = MathTex(r"D = b^2 - 4ac", color=TXT).scale(0.9)
        disc2 = MathTex(r"D = (-10)^2 - 4(1)(175)", color=TXT).scale(0.9)
        disc3 = MathTex(r"D = 100 - 700 = -600", color=IGNITION).scale(1.0)
        neg  = MathTex(r"D < 0", color=REDX).scale(1.2)
        VGroup(disc, disc2, disc3, neg).arrange(DOWN, buff=0.45).shift(DOWN*0.3)

        with self.voiceover(text="Now here's the clever move. Before we solve anything... "
                                 "just check the discriminant. b squared, minus four a c.") as t:
            self.play(Write(disc), run_time=min(1.6, t.duration))
            self.wait(max(0.1, t.duration - 1.6))
        with self.voiceover(text="That's minus ten, squared... minus four, times one, times one seventy-five.") as t:
            self.play(TransformFromCopy(disc, disc2), run_time=min(1.6, t.duration))
            self.wait(max(0.1, t.duration - 1.6))
        with self.voiceover(text="A hundred... minus seven hundred. That is negative six hundred.") as t:
            self.play(TransformFromCopy(disc2, disc3), run_time=min(1.6, t.duration))
            self.wait(max(0.1, t.duration - 1.6))
        with self.voiceover(text="Negative. The discriminant is negative — and that... changes everything. "
                                 "There are no real solutions. None. Which means there is no moment in time "
                                 "where these two are ever in the same place.") as t:
            self.play(Write(neg), run_time=min(1.2, t.duration))
            self.play(Flash(neg, color=REDX, flash_radius=1.0), run_time=0.8)
            self.wait(max(0.1, t.duration - 2.0))
        self.play(FadeOut(disc), FadeOut(disc2), FadeOut(disc3), FadeOut(neg), FadeOut(quad))

        # ================= 06 — GRAPH PAYOFF + ANSWER =================
        set_prog(6)
        ax = Axes(
            x_range=[0, 20, 5], y_range=[0, 650, 200],
            x_length=8.5, y_length=4.2,
            axis_config={"color": MUTED, "include_tip": True,
                         "font_size": 22, "stroke_width": 2},
            tips=True,
        ).to_edge(DOWN, buff=0.7)
        xlab = Text("t (s)", font=MONO, color=MUTED).scale(0.35).next_to(ax.x_axis, RIGHT, buff=0.15)
        ylab = Text("x (m)", font=MONO, color=MUTED).scale(0.35).next_to(ax.y_axis, UP, buff=0.15)

        g1 = ax.plot(lambda t: 10*t + t**2, x_range=[0, 20], color=CYAN, stroke_width=4)
        g2 = ax.plot(lambda t: 20*(t-5) + 0.5*(t-5)**2, x_range=[5, 20], color=EMBER, stroke_width=4)
        g1l = MathTex(r"x_1", color=CYAN).scale(0.7).next_to(ax.c2p(20, 600), UR, buff=0.05)
        g2l = MathTex(r"x_2", color=EMBER).scale(0.7).next_to(ax.c2p(20, 412), RIGHT, buff=0.1)

        with self.voiceover(text="Let's actually see it. Particle one's path, in cyan... "
                                 "and particle two's, in orange — joining in at five seconds.") as t:
            self.play(Create(ax), FadeIn(xlab), FadeIn(ylab), run_time=1.6)
            self.play(Create(g1), run_time=min(1.6, max(0.6, t.duration - 2.2)))
            self.play(Create(g2), Write(g1l), Write(g2l), run_time=1.2)
            self.wait(0.2)

        # the gap that never closes
        gap = always_redraw(lambda: DashedLine(
            ax.c2p(20, 20*(15)+0.5*15**2), ax.c2p(20, 10*20+20**2),
            color=IGNITION, stroke_width=3))
        gaplab = Text("gap keeps widening", font=MONO, color=IGNITION).scale(0.38)
        gaplab.next_to(ax.c2p(15, 500), UP, buff=0.1)

        with self.voiceover(text="Look closely — the orange curve never crosses the cyan one. "
                                 "At five seconds, yes, they share the very same speed. "
                                 "But particle one's stronger acceleration just... pulls away. "
                                 "And that gap? It only widens.") as t:
            self.play(Create(gap), FadeIn(gaplab), run_time=1.4)
            self.wait(max(0.1, t.duration - 1.4))

        self.play(FadeOut(gaplab), FadeOut(gap),
                  ax.animate.scale(0.8).to_edge(DOWN, buff=0.5),
                  VGroup(g1, g2, g1l, g2l).animate.scale(0.8),
                  run_time=0.8)

        ans = VGroup(
            Text("ANSWER", font=MONO, color=MUTED).scale(0.4),
            Text("(D)  never catches", font=MONO, color=IGNITION, weight=BOLD).scale(0.6),
        ).arrange(DOWN, buff=0.2).to_edge(UP, buff=0.9)
        box = SurroundingRectangle(ans[1], color=IGNITION, buff=0.25, corner_radius=0.1)

        with self.voiceover(text="So our answer... is D. The second particle — never catches the first.") as t:
            self.play(FadeIn(ans, shift=DOWN*0.2), run_time=1.0)
            self.play(Create(box), run_time=0.8)
            self.wait(max(0.3, t.duration - 1.8))
        self.wait(0.6)
