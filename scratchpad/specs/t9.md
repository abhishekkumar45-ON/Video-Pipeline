## Question
One mole of helium at (P₁, V₁) expands ISOTHERMALLY to 4V₁, then ADIABATICALLY to 32V₁. If the
work done in the two steps satisfies W_iso / W_adia = f · ln2, find f.

## KNOWN ANSWER (anchor to this): f = 16/9 ≈ 1.78

## Concept-first
- Isothermal work: W_iso = nRT ln(V_f/V_i).
- Adiabatic (helium, monatomic): γ = 5/3, γ−1 = 2/3; T V^(γ−1) = const.
- Adiabatic work: W_adia = nR(T_i − T_f)/(γ−1).

## LIVE VISUAL
Draw a P–V plane: an isotherm segment V₁ → 4V₁ (EMBER) then an adiabatic segment 4V₁ → 32V₁
(IGNITION, steeper). Mark the three volumes. Optional live V readout as a dot travels. Dock after.

## STEP RAIL (LAW → SUBSTITUTION → RESULT)
1. Isothermal (at T₁): `W_iso = nRT₁ ln(4V₁/V₁) = RT₁ ln4 = 2 R T₁ ln2`.
2. Temperature after the adiabatic step (from (T₁, 4V₁) to 32V₁, ratio 8), γ−1 = 2/3:
   `T_f = T₁ (4V₁/32V₁)^(2/3) = T₁ (1/8)^(2/3) = T₁/4`.
3. Adiabatic work: `W_adia = nR(T₁ − T_f)/(γ−1) = R(T₁ − T₁/4)/(2/3) = (3/2)R(3T₁/4) = (9/8) R T₁`.
4. Ratio: `W_iso/W_adia = (2 R T₁ ln2)/((9/8) R T₁) = (16/9) ln2` → **f = 16/9 ≈ 1.78**
   (results rail: `f = 16/9 \approx 1.78`, CORRECT green).

Answer lock-in: **f = 16/9 ≈ 1.78**.
